# Самописный поисковый агент на основе DeepAgents From Scratch

## Requirements
- [high] Использование DeepAgents From Scratch: Весь код агента должен быть написан с нуля, без использования готовых моделей из сторонних библиотек. Необходимо реализовать архитектуру, обучение и инференс с помощью DeepAgents.
- [high] Обучение модели: Модель должна быть обучена на реальном датасете (например, Wikipedia, Common Crawl) и достигать минимум 70% точности по метрике MAP.
- [normal] Интеграция с веб‑интерфейсом: Создать REST API для взаимодействия с фронтендом, позволяющий отправлять запросы и получать результаты в формате JSON.
- [normal] Документация и тесты: Подготовить README с инструкциями по запуску, обучению и тестированию. Написать unit‑тесты для ключевых компонентов.
